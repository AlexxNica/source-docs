KeyboardShortcut.makeKey(keyCode, modifiers)

Creates a number encoding keyCode in the lower 8 bits and modifiers mask in the higher 8 bits. It is useful for matching pressed keys.

Parameters
keyCode(number|string)

The code of the key, or a character "a-z" which is converted to a keyCode value.

modifiersnumber=

Optional list of modifiers passed as additional parameters.
